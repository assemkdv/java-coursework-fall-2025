# Sum of Digits of a Three-Digit Number

This program reads a **three-digit integer** from the user—positive or negative—and calculates the **sum of its digits**. The program uses arithmetic operations and `Math.abs()` to correctly handle negative numbers.

---

## Objective

- Practice extracting digits using division (`/`) and modulus (`%`)
- Use `Math.abs()` to remove negative signs before digit processing
- Work with console input using `Scanner`
- Strengthen understanding of integer manipulation in Java

---

## Sample Output 

```text

Enter a three-digit number:
-123
Sum of digits: 6

Enter a three-digit number:
456
Sum of digits: 15
